# Gutters

Wrap content with `<Gutters />` to apply horizontal margins to it.

Typically you'll want to render one of these at the top level of a page, so that content doesn't jam up directly against the viewport edge.


```tsx
import { Gutters } from "@cy7/design-system";
```

## Laying out a page

Consider using `<Gutters />` in conjunction with [the `<MaxWidth />` component](/design-system-layout-maxwidth--page) for a standard centred page layout.

<Canvas withSource="open">
  <Story name="With MaxWidth">

      <Gutters>
        <Stack space="2">
          <MaxWidth>
            <Box css={{ border: "dashed 2px $candyfloss", height: "120px"}} />
          </MaxWidth>
          <MaxWidth width="bestForText">
            <h1>Lorem ipsum dolor sit amet</h1>
            <p>
              Nulla at volutpat diam ut venenatis tellus in. Vel elit scelerisque mauris pellentesque pulvinar. Pharetra vel turpis nunc eget lorem dolor sed viverra ipsum. Nunc sed augue lacus viverra vitae congue eu consequat ac. Massa tincidunt dui ut ornare lectus sit.
            </p>
          </MaxWidth>
        </Stack>
      </Gutters>
    </Box>
  </Story>
</Canvas>
