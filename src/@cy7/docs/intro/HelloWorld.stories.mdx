## Getting started

All the code needed to run and deploy the site lives in [a single open source repository on Gitlab](https://gitlab.com/cycleseven/cy7).

1. Ensure you have a working copy of `git` on your development machine.
1. Clone [the project](https://gitlab.com/cycleseven/cy7).
1. Follow the README instructions to get up and running.

## Tech stack

[cy7.io](https://cy7.io) is a static site, written in [TypeScript](https://www.typescriptlang.org/), built using [Gatsby](https://www.gatsbyjs.com/), and styled with [Stitches](https://stitches.dev/).

These docs are built with [Storybook](https://storybook.js.org/).
