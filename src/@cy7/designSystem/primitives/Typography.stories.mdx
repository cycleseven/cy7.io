# Typography

Typography styles are global (generated by [Typography.js](https://kyleamathews.github.io/typography.js/)). Use the standard HTML tags like `h1`, `p`, etc.

## The typography theme object

[Typography.js](https://github.com/KyleAMathews/typography.js) returns a config object, which is made available in `theme.typography`.

This allows access to config values like `baseLineHeight` and `scaleRatio` via the theme.

An especially useful utility is the [`rhythm()` function](https://github.com/KyleAMathews/typography.js/blob/master/docs/what.md#base-all-other-spacings-off-the-rhythm-unit) which returns `rem` values based on the global line height. You can use this function to pick consistent vertical spacing values, even for non-typography components.

```
const ComponentUsingTypography = styled.div`
  padding-top: ${props => props.theme.typography.rhythm(2)};
`;
```
