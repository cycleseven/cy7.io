# Scales

> **Note:** experimental + subject to change.

Scales provide base values for CSS sizes to encourage consistency. You can access scales via the [Styled Components theme](https://www.styled-components.com/docs/advanced#theming) like this:

```
const ComponentUsingScales = styled.div`
  border-radius: ${props => props.theme.scales.borderRadius()};
`;
```

You can pass a multiplier to a scale function. For example, to use a doubled border radius size, do this:

```js
props.theme.scales.borderRadius(2);
```

## Available scales

* `borderRadius`

## Vertical spacing

Vertical rhythm is an important scale that's missing here. It's defined as part of the typography instead! Use `theme.typography.rhythm(n)`.
