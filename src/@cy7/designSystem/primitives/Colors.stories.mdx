# Colors

All these colors are available via [the `theme` prop in Styled Components](https://www.styled-components.com/docs/advanced#theming).

> If the `theme` isn't available, make sure the component tree is wrapped `<Cy7ThemeProvider />`.

```
const ComponentUsingColors = styled.div`
  background-color: ${props => props.theme.colors.paper()};
  color: ${props => props.theme.colors.obsidian()};
`;
```

For use outside of Styled Components, import the colors directly.

```js
import { colors } from "designPrimitives";
const { paper, candyfloss } = colors;
```
