# Getting started

Wrap the root component in `<Cy7ThemeProvider />`. The theme provider applies global styles to the document, and makes design primitives available via the [Styled Components theme](https://www.styled-components.com/docs/advanced#theming).

```
const ComponentUsingTheme = styled.div`
  color: ${props => props.theme.colours.obsidian}
`;
```
