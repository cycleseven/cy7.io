# Colours

All these colours are available via [the `theme` prop in Styled Components](https://www.styled-components.com/docs/advanced#theming).

> If the `theme` isn't available, make sure the component tree is wrapped `<Cy7ThemeProvider />` as detailed in **Getting started**.

```
const ComponentUsingColours = styled.div`
  background-color: ${props => props.theme.colours.paper}
  color: ${props => props.theme.colours.obsidian}
`;
```

For use outside of Styled Components, import the colours directly.

```js
import { colours } from "designPrimitives";
const { paper, onyx } = colours;
```
